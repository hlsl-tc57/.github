# Ecma TC57 - HLSL

[Ecma Technical Committee 57][TC57] is the committee to standardize the High
Level Shading Language (HLSL).

## Navigating our GitHub

All primary work for the technical committee occurs in the [main GitHub
repository][MainRepo].

To contribute please review the [Contributing Documentation][Contrib].

To follow the activities of the TC you may find approved meeting minutes in the
[docs/MeetingMinutes][ApprovedMinutes] folder. Agendas for upcoming meetings and
unapproved minutes are [Pull Requests with the `meetings` label][MeetingsLabel].
